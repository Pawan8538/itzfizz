# ItzFizz Landing Page

This repository contains the front-end implementation for the ItzFizz landing page, re-designed to precisely match a premium Figma design. 

## Tech Stack

* Framework: Next.js (App Router)
* Library: React 
* Styling: Tailwind CSS v4
* Animation: GSAP (GreenSock Animation Platform)

## Features and Implementation Details

* Hero Animation: Features a custom GSAP animation where the centerpiece car element exits smoothly (70 percent past the screen edge).
* Dynamic Typography: The "WELCOME ITZFIZZ" headline is vertically centered with increased letter spacing for visual impact.
* Staggered Animations: Stat cards and feature sections animate into view dynamically on page load to enhance user engagement.
* Clean Codebase: Built strictly using Vanilla JavaScript and JSX, ensuring minimal dependencies and straightforward components.

## Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open http://localhost:3000 with your browser to see the resul.
